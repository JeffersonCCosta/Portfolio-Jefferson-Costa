Tenho perfil analítico, aprendizado rápido e forte orientação a resultados, buscando sempre otimização de processos e geração de valor por meio da tecnologia. Estou em constante evolução técnica, com foco em desenvolvimento de software, backend e soluções baseadas em dados.

Projetos Atuais

Sistema de Estacionamento

Projeto em desenvolvimento para um sistema de gerenciamento de estacionamento, utilizando Java com Spring Boot no backend e HTML, CSS e JavaScript no frontend. O sistema tem como objetivo cadastrar veículos, registrar entradas e saídas e disponibilizar consultas em tempo real, visando eficiência e praticidade na gestão.

Sistema de Controle Financeiro

Sistema web completo para gerenciamento de finanças pessoais, desenvolvido com Java (Spring Boot) no backend e HTML, CSS e JavaScript puro no frontend. A aplicação permite cadastro e autenticação de usuários, controle de receitas e despesas, categorização de lançamentos e visualização de dados em dashboard interativo com gráficos e totais consolidados. O projeto segue arquitetura REST, integração via API, organização em camadas (controller, service, repository) e boas práticas de versionamento com Git/GitHub.
